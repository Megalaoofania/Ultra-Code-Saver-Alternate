Ultra Code Saver

Welcome to the Ultra Code Saver, pretty much what we are is like copy code from a website, show different parts if the code has been updated since the last request (if the website was ever requested at all) and save onto here. Just like the wayback machine except it's only code and no website! (atleast not yet, maybe in a furture release wink wink nudge nudge).

To add/update a website please contact me at Mega#3764 on Discord or on the issues tab of this project.
